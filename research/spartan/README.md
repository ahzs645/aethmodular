# SPARTAN network — data inventory, coverage, and bridges

This folder is the durable record of what data the SPARTAN network publishes,
how the public release relates to the HIPS optical-absorption file shared
separately, and how each site connects across the available sources.

All artifacts here were generated by scripts under
[`scripts/pipelines/spartan_*.py`](../../scripts/pipelines/). Raw downloads
live under `data/spartan/raw/` and `data/drive_bridge/Spartan/`, both
git-ignored.

---

## TL;DR — what we know

- **40 distinct sites** appear across all sources combined.
  - **37** are in the public FTP (http://data.spartan-network.org/).
  - **27** are in the HIPS Drive file.
  - **37** are in the Drive site lookup.
  - See [`master_connections.md`](master_connections.md) and the
    [`connection_matrix.png`](figures/connection_matrix.png) heatmap for the
    full site-by-source grid.

- **Public dataset** = 230 CSVs, ~939,300 rows, ~166 MB on disk, across
  **3 products** and **9 subproducts**:

  | Product | Subproducts | What it is |
  |---|---|---|
  | `FilterBased` | `ChemSpecPM10`, `ChemSpecPM25`, `ReconstrPM25` | Filter chemistry (PM mass, BC, EC/OC, OM, IC ions, metals panel) and IMPROVE-style reconstruction |
  | `NephelProcd` | `Daily/Hourly Sca PM10`, `Daily/Hourly Sca PM25` | Nephelometer scattering (back/total at R, G, B) |
  | `TimeResPM25` | `Daily/Hourly EstPM25` | Estimated PM2.5 mass from filter-anchored nephelometer |

- **Sampling cadence** confirmed by interval analysis:
  - FilterBased PM2.5 = SPARTAN nominal **1-in-9-days** (median Δ ≈ 216 h).
  - FilterBased PM10 is much sparser (median Δ ≈ 74 days; ~8% of nominal).
  - Nephelometer products = **hourly** (median Δ = 1 h) but typical fill is
    only **40-50%** because the SPARTAN processing screens out periods that
    fail RH/scattering QC.
  - Date range: 2013-06-28 → 2026-04-26 (ignoring 2 bad year-2166/2167 typo
    rows in the ILNZ source files — see Caveats).

- **HIPS Drive folder**
  (`1YVmkYP_0pzs5TQwwbcTQi7gEJ-rTl9LZ`) carries **3,178 sample filters +
  916 blanks** with raw optical absorption (`Fabs`), uncertainty, MDL, and
  the underlying T1/R1/t/r/tau. Coverage starts ~2022-04.

- **Critical finding**: the public `BC PM2.5` column **is the HIPS-derived
  BC** — at every site, `BC_public = HIPS Fabs / 10` (slope = 10 m²/g,
  intercept ≈ 0, R² ≈ 1.000). They are not independent measurements.
  Implication: for absorption-based work, use HIPS `Fabs` directly so you
  can vary MAC if you want; don't treat public BC as a check on HIPS.
  See [`HIPS_BRIDGE.md`](HIPS_BRIDGE.md).

- **Highest-coverage sites** (composite of completeness × breadth):
  CADO, AUMN, USNO, CASH, KRUL, CHTS, SGSU, MXMC, CALE, AEAZ, CAHA.
  CASH is by far the largest by raw observation volume (~150k rows).
  See [`top_sites.csv`](top_sites.csv).

- **2025 commissioning sites** that exist in HIPS but not in the public FTP
  yet: **ETBI** (Bishoftu, Ethiopia — sister to ETAD, 26 filters from Oct
  2025) and **USSL** (St Louis — 42 filters from Aug 2025).

## How to reproduce the analysis

All scripts assume the current working directory is the repo root.

```bash
# 1. Crawl SPARTAN, download every CSV, and write the inventory + REPORT.md
python scripts/pipelines/spartan_pull_and_summarize.py
# (uses --skip-download to reuse already-cached files)

# 2. Per-(site, subproduct) coverage and sampling-interval stats
python scripts/pipelines/spartan_coverage_analysis.py

# 3. Timeline plots, heatmaps, top-site ranking
python scripts/pipelines/spartan_coverage_plots.py

# 4. Split rankings, ETAD parameter breakdown, world map
python scripts/pipelines/spartan_extras.py

# 5. Bridge to HIPS Drive folder (pulls 3 files via gdown the first time)
python scripts/pipelines/spartan_hips_bridge.py

# 6. The master connection matrix across every source
python scripts/pipelines/spartan_master_connections.py
```

Dependencies (install if missing): `pandas`, `requests`, `matplotlib`,
`tabulate` (for the markdown report), `openpyxl` (for the Drive lookup
xlsx), `gdown` (for the Drive folder), `cartopy` (for the world map).

## How to *use* this data downstream

### Pick a site to work with

Open [`top_sites.csv`](top_sites.csv). Sort by `score` for "broadest +
fullest data", or by `total_samples` for "most observations". Then look at
[`master_connections.csv`](master_connections.csv) to see whether the site
also has a HIPS row — if you need Fabs, you need a site in both.

The best 9-subproduct + HIPS sites are: **CASH, CHTS, AEAZ, AUMN, ZAPR,
KRUL, MXMC, USNO, KRSE, ILNZ, CAHA**. CADO has nine subproducts but is
*not* in HIPS.

### Load a single site's filter chemistry into pandas

```python
import pandas as pd
df = pd.read_csv(
    "data/spartan/raw/FilterBased/ChemSpecPM25/FilterBased_ChemSpecPM25_ETAD.csv",
    skiprows=3,           # 3 comment lines, then the header
    low_memory=False,
)
# Tidy: each row = (filter, parameter); pivot if you want a filter-keyed wide frame
wide = df.pivot_table(
    index=["Filter_ID", "Start_Year_local", "Start_Month_local", "Start_Day_local"],
    columns="Parameter_Name", values="Value", aggfunc="mean",
)
```

### Join public chemistry to HIPS Fabs by filter ID

```python
import pandas as pd
hips = pd.read_csv("data/drive_bridge/Spartan/SPARTAN_HIPS_Batch1-51.v2.csv")
hips["FilterId_base"] = hips["FilterId"].str.replace(r"-(\d)$", "", regex=True)

pub = pd.read_csv("data/spartan/raw/FilterBased/ChemSpecPM25/FilterBased_ChemSpecPM25_ETAD.csv", skiprows=3)
pub["FilterId_base"] = pub["Filter_ID"].str.replace(r"-(\d)$", "", regex=True)

merged = pub.merge(
    hips[["FilterId_base", "Fabs", "MDL", "Uncertainty"]].drop_duplicates("FilterId_base"),
    on="FilterId_base", how="left",
)
```

### Recompute BC at a different MAC

Public BC is hard-coded to MAC = 10. If you want a different MAC:

```python
bc_custom_mac = hips["Fabs"] / mac_choice   # mac_choice in m^2/g
```

### Build a site-level summary for one country

```python
master = pd.read_csv("research/spartan/inventory/master_connections.csv")
chinese = master[master["lookup_name"].str.contains("China", na=False)]
# → only CHTS in this dataset
```

## Caveats and known data quirks

1. **ILNZ bad-year typo**. The TimeResPM25 ILNZ files contain ~155 rows
   with year `2166`/`2167`. The coverage script filters these out (years
   restricted to 2010-2030). If you load those files directly, watch out.

2. **`FilterBased` completeness is normalised to a 1-in-9-days cadence.**
   A handful of newer high-cadence sites (CLTA, CLST, INDH, INJA, USPA)
   sample more often than that, so their raw completeness exceeds 100% and
   is capped at 100 for the rankings. The actual cadence at those sites is
   closer to 3-5 days.

3. **Public `BC PM2.5` is a derived product**, not an independent
   measurement. Treat it as `Fabs / 10`.

4. **Nephelometer "completeness" of 40-50% is normal**, not a problem with
   the data. The SPARTAN processing only retains hours that pass an RH /
   scattering QC. High raw nephelometer fill (> 80%) only happens at the
   best-instrumented sites (AUMN, USNO).

5. **`FilterId` formats vary slightly** across files — some carry a
   trailing replicate suffix (`SITE-NNNN-R`), others use the base
   `SITE-NNNN`. The bridge script normalises by stripping the trailing
   `-R`. Replicate `-7` is reserved for **field blanks** and shouldn't be
   matched to a real chemistry filter.

6. **Three sites are in the public FTP but missing from the Drive
   lookup**: `ARCB`, `USBO`, `USMC`. They look legacy/decommissioned. Lat/lon
   for those still works via the public CSVs themselves (they carry
   `Latitude`/`Longitude` columns).

7. **HIPS misses 12 public sites**: CADO, CAKE, CALE, CODC, INKA, NGIL,
   PHMO, SGSU, VNHN, ARCB, USBO, USMC. If you need optical absorption at
   any of those, the Drive folder I have doesn't cover them — there may be
   another batch elsewhere.

8. **2025 commissioning sites** (`ETBI`, `USSL`, `USBA`) won't appear in
   the public FTP until the next data version. Use HIPS for now.
