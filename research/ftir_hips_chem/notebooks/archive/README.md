# Archived Notebooks

This folder contains notebooks kept for provenance, not active extension.

## `executed/`

Executed copies of active notebooks. Keep the non-`_executed` notebook in
`research/ftir_hips_chem/` as the source of truth and regenerate outputs from
there when needed.

## `scratch/`

Scratch, copy, or legacy helper notebooks that have been superseded by the
modular analysis workflow. These may still be useful for recovering old ideas,
but new work should start from `Example_Modular_Analysis.ipynb` or an existing
active notebook at the research root.
